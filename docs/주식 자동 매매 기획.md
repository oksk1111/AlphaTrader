# 📁 문서 1. [기획서] Project US-ETF-Sniper

## 1. 프로젝트 개요

- **프로젝트 명:** US-ETF-Sniper (미국 주식 레버리지 ETF 자동매매 봇)
- **목표:** 나스닥/반도체 상승장에서 **3배 레버리지(3x)**로 수익을 극대화하되, AI로 하락장을 감지하여 **MDD(최대 낙폭)를 방어**한다.
- **핵심 대상:**
    1. **TQQQ** (ProShares UltraPro QQQ) - 나스닥 100 지수 3배 추종
    2. **SOXL** (Direxion Daily Semiconductor Bull 3X) - 반도체 지수 3배 추종
- **운영 시간:** 미국 정규장 시간 (한국 시간 23:30 ~ 06:00, 서머타임 시 1시간 당겨짐)

## 2. 투자 전략 (The Strategy)

전통적인 퀀트 전략인 **'변동성 돌파(VBO)'**에 **'AI 거시경제 필터'**를 장착한 하이브리드 전략입니다.
**v2.0 업데이트**: DCA(분할매수) 모드 추가로 안정적인 수익 추구.

### 전략 모드 선택

| 모드 | 설명 | 적합한 시장 |
|------|------|------------|
| **VBO (변동성 돌파)** | 목표가 돌파 시 매수, 장 마감 매도 | 강한 상승장 |
| **DCA (분할매수)** | 매일 정해진 금액 분할 매수 | 횡보장/장기 투자 |
| **HYBRID (하이브리드)** | 추세 상승 시 VBO, 그 외 DCA | 모든 시장 |

### Step 1. [Trend Filter] 추세 판단 (장 시작 전)

**"상승장(Bull Market)에서만 3배 레버리지를 탄다."**

- **조건:** 현재 주가가 **20일 이동평균선(20MA)** 위에 있는가?
- **Action:**
    - 위(True): 매수 진입 준비 (Risk-On).
    - 아래(False): DCA 모드에서는 소량 매수, VBO 모드에서는 쉼.

### Step 2. [Entry Trigger] 진입 타점

#### VBO 모드 (변동성 돌파)
- **공식:** 매수 주문가 = $Today Open + (Yesterday Range \times K)$
    - $Yesterday Range = (Yesterday High - Yesterday Low)$
    - $K = 0.5$ (노이즈 필터링 상수)
- **의미:** 장 시작 후 가격이 시가 대비 일정 폭 이상 상승하면 추격 매수.

#### DCA 모드 (분할매수)
- **공식:** 매일 고정 금액 투자
    - 예: 하루 $50 또는 가용 자금의 5%
- **타이밍:** 장 시작 30분 후 자동 매수 (시가 안정화 후)
- **장점:** 시장 타이밍 부담 없음, 평균 매입단가 하락 효과

#### HYBRID 모드 (하이브리드)
- **추세 상승 (20MA 위):** VBO 전략 실행
- **추세 횡보/하락 (20MA 아래):** DCA 전략 실행 (소량)

### Step 3. [AI Macro Filter] 재료 검증 (진입 직전)

**"기술적 신호가 떴지만, 혹시 오늘 파월 의장이 금리 인상 발언을 했는가?"**

- **역할:** 차트는 매수라고 하지만, 악재 뉴스가 터진 경우를 AI가 걸러냄.
- **LLM Prompt:** 미국 주요 경제 뉴스(CNBC, [Investing.com](http://investing.com/) 헤드라인)를 Gemini에게 입력.
    - *"Analyze the current US market sentiment. Is there any 'Panic Selling' risk today due to Fed or CPI data? Answer YES/NO."*
- **Action:** 
    - VBO 모드: AI가 "Panic Risk"라고 판단하면 매수 취소.
    - DCA 모드: AI 경고 시 매수량 50% 감소 (완전 중단 X)

### Step 4. [Constraints & Fallback] 제약 사항 및 대응 (Safe Mode)

**"법적/자금적 제약으로 레버리지(3x) ETF 매수가 불가능한 경우"**

- **상황:** 레버리지 ETF 교육은 이수했으나, **기본 예탁금(1,000만원)** 기준을 충족하지 못한 경우.
- **제약:** 증권사 API에서 3배수 ETF(`SOXL`, `TQQQ` 등) 매수 주문 시 "예탁금 부족(APBK1681)" 오류 발생.
- **대응 (Safe Mode):** 자동으로 **1배수 정방향 자산**으로 타겟을 변경하여 매매를 지속함.
    - `SOXL` (3x) $\rightarrow$ `SOXX` (1x 반도체) 또는 `NVDA`
    - `TQQQ` (3x) $\rightarrow$ `QQQ` (1x 나스닥)
    - `BITX` (2x) $\rightarrow$ `IBIT` (1x 비트코인)
    - `CONL` (2x) $\rightarrow$ `COIN` (코인베이스)

### Step 5. [Exit Rule] 청산 전략

#### Day Trading 모드
- **손절(Stop Loss):** 매수가 대비 **3%** 도달 시 즉시 시장가 매도. (타협 없음)
- **익절(Take Profit):** 트레일링 스탑 적용 (고점 대비 2~3% 하락 시 이익 실현).
- **종가 청산(Time Cut):** 장 마감 10분 전(05:50) 전량 매도. (오버나이트 리스크 제거)

#### Swing/Hold 모드 (DCA 포함)
- **추세 이탈 매도:** 종가가 20MA 아래로 이탈 시 다음 날 매도
- **리밸런싱:** 월 1회 포트폴리오 점검, 비중 조정
- **손절:** 개별 종목 -10% 손실 시 매도 (장기 투자 손절선 완화)

### 3.2. AI 분석기 (`modules/gemini_analyst.py`)

Gemini에게 미국 시장 상황을 영어로 물어봅니다.

```python
import google.generativeai as genai

class GeminiAnalyst:
    def __init__(self, api_key):
        genai.configure(api_key=api_key)
        self.model = genai.GenerativeModel('gemini-1.5-flash')

    def check_market_sentiment(self, news_text):
        prompt = f"""
        Act as a aggressive stock trader.
        Here are the latest news headlines regarding US Tech Market & Fed:
        {news_text}

        Critical Check:
        1. Is there any MAJOR crash signal (e.g. War, Unexpected Rate Hike)?
        2. Is the sentiment predominantly Fear?

        Reply with JSON:
        {{
            "risk_level": "HIGH" or "LOW",
            "can_buy": boolean,
            "reason": "short summary"
        }}
        """
        response = self.model.generate_content(prompt)
        return response.text # JSON 파싱 후 can_buy 리턴

```
